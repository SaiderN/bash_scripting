## –¢–∏–ø–∏—á–Ω—ã–π bash-—Å–∫—Ä–∏–ø—Ç –∏–º–µ–µ—Ç –ø—Ä–æ—Å—Ç—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É:

**(–¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ)**
### –¢–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –≤ Bash
Bash –ø–æ —Å–≤–æ–µ–π —Å—É—Ç–∏ ‚Äî —ç—Ç–æ –æ–±–æ–ª–æ—á–∫–∞ –¥–ª—è –∑–∞–ø—É—Å–∫–∞ –∫–æ–º–∞–Ω–¥, –∏ –æ–Ω–∞ **–Ω–µ –∏–º–µ–µ—Ç —Å—Ç—Ä–æ–≥–æ–π —Ç–∏–ø–∏–∑–∞—Ü–∏–∏** –∫–∞–∫, –Ω–∞–ø—Ä–∏–º–µ—Ä, C++ –∏–ª–∏ Java. –í—Å–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é —Ö—Ä–∞–Ω—è—Ç—Å—è –∫–∞–∫ —Å—Ç—Ä–æ–∫–∏ (strings). –û–¥–Ω–∞–∫–æ –∫–æ–Ω—Ç–µ–∫—Å—Ç –∏—Ö –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è (–≤ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏—è—Ö, —Å—Ä–∞–≤–Ω–µ–Ω–∏—è—Ö) –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫ –æ–Ω–∏ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä—É—é—Ç—Å—è.

**–û—Å–Ω–æ–≤–Ω—ã–µ "—Ç–∏–ø—ã" –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ:**
1.  **–°—Ç—Ä–æ–∫–∞ (String):** –ó–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é. `name="Alice"`, `path="/usr/bin"`
2.  **–¶–µ–ª–æ–µ —á–∏—Å–ª–æ (Integer):** –î–ª—è –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π. –°–æ–∑–¥–∞—ë—Ç—Å—è –Ω–µ –æ–±—ä—è–≤–ª–µ–Ω–∏–µ–º, –∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –≤ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–æ–º –∫–æ–Ω—Ç–µ–∫—Å—Ç–µ: `count=5` (—Ö—Ä–∞–Ω–∏—Ç—Å—è –∫–∞–∫ —Å—Ç—Ä–æ–∫–∞ "5", –Ω–æ –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–∞–∫ —á–∏—Å–ª–æ).
3.  **–ú–∞—Å—Å–∏–≤ (Array):** –ò–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ –∑–Ω–∞—á–µ–Ω–∏–π. `files=(file1.txt file2.txt)`
4.  **–ê—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω—ã–π –º–∞—Å—Å–∏–≤ (Associative Array):** –ü–∞—Ä—ã "–∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ" (–∫–∞–∫ —Å–ª–æ–≤–∞—Ä—å). `declare -A user=([name]="John" [id]=123)`
5.  **"–ë—É–ª–µ–≤—ã" –∑–Ω–∞—á–µ–Ω–∏—è:** –ü—Ä—è–º—ã—Ö `true`/`false` –Ω–µ—Ç. –£—Å–ø–µ—Ö –∏–ª–∏ –Ω–µ—É–¥–∞—á–∞ –æ–ø–µ—Ä–∞—Ü–∏–∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è **–∫–æ–¥–æ–º –≤–æ–∑–≤—Ä–∞—Ç–∞** (`0` = –∏—Å—Ç–∏–Ω–∞/—É—Å–ø–µ—Ö, **–ª—é–±–æ–µ –Ω–µ-–Ω—É–ª–µ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ** = –ª–æ–∂—å/–æ—à–∏–±–∫–∞). –î–ª—è –ø—Ä–æ–≤–µ—Ä–æ–∫ —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –∫–æ–º–∞–Ω–¥—ã `true` (–≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç 0) –∏ `false` (–≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç 1).

**(–¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ)**
### –û–±—ä—è–≤–ª–µ–Ω–∏–µ –∏ –∞—Ç—Ä–∏–±—É—Ç—ã –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö
–•–æ—Ç—è —è–≤–Ω–æ –æ–±—ä—è–≤–∏—Ç—å —Ç–∏–ø `int` –Ω–µ–ª—å–∑—è, –∫–æ–º–∞–Ω–¥–∞ `declare` (–∏–ª–∏ –µ—ë —Å–∏–Ω–æ–Ω–∏–º—ã `typeset`) –ø–æ–∑–≤–æ–ª—è–µ—Ç –∑–∞–¥–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –æ—Å–æ–±—ã–µ *–∞—Ç—Ä–∏–±—É—Ç—ã*, –≤–ª–∏—è—é—â–∏–µ –Ω–∞ –µ—ë –ø–æ–≤–µ–¥–µ–Ω–∏–µ.

```bash
#!/bin/bash

# -i : –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –∫–∞–∫ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ (integer) –≤ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏—è—Ö.
declare -i number
number="10"   # –°—Ç—Ä–æ–∫–∞ "10" –±—É–¥–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∞ –≤ —á–∏—Å–ª–æ 10
number="hello" # –ï—Å–ª–∏ –ø—Ä–∏—Å–≤–æ–∏—Ç—å –Ω–µ-—á–∏—Å–ª–æ, –æ–Ω–æ —Å—Ç–∞–Ω–µ—Ç 0
echo $number   # –í—ã–≤–µ–¥–µ—Ç 0

# -r : –°–¥–µ–ª–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é —Ç–æ–ª—å–∫–æ –¥–ª—è —á—Ç–µ–Ω–∏—è (readonly/–∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞).
declare -r PI=3.14159
# PI=3.14 # –≠—Ç–∞ —Å—Ç—Ä–æ–∫–∞ –≤—ã–∑–æ–≤–µ—Ç –æ—à–∏–±–∫—É: "readonly variable"

# -a : –û–±—ä—è–≤–∏—Ç—å –∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–∞—Å—Å–∏–≤.
declare -a colors=("red" "green" "blue")
echo ${colors[1]} # –í—ã–≤–µ–¥–µ—Ç "green" (–∏–Ω–¥–µ–∫—Å–∞—Ü–∏—è —Å 0)

# -A : –û–±—ä—è–≤–∏—Ç—å –∞—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω—ã–π –º–∞—Å—Å–∏–≤ (—Ç—Ä–µ–±—É–µ—Ç Bash –≤–µ—Ä—Å–∏–∏ 4.0+).
declare -A user
user[name]="Bob"
user[age]=25
echo ${user[name]} # –í—ã–≤–µ–¥–µ—Ç "Bob"

# -x : –≠–∫—Å–ø–æ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –≤ –æ–∫—Ä—É–∂–µ–Ω–∏–µ –¥–ª—è –¥–æ—á–µ—Ä–Ω–∏—Ö –ø—Ä–æ—Ü–µ—Å—Å–æ–≤.
declare -x MY_TOOL_PATH="/opt/mytool"
# –¢–µ–ø–µ—Ä—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è MY_TOOL_PATH –±—É–¥–µ—Ç –¥–æ—Å—Ç—É–ø–Ω–∞ –≤ –ª—é–±–æ–º —Å–∫—Ä–∏–ø—Ç–µ, –∑–∞–ø—É—â–µ–Ω–Ω–æ–º –∏–∑ —ç—Ç–æ–≥–æ.
```
### –í—ã–≤–æ–¥ —Å –∫–æ–º–∞–Ω–¥–æ–π echo:
–ï—Å–ª–∏ —Ö–æ—Ç–∏–º –ø—Ä–æ—Å—Ç–æ –≤—ã–≤–µ—Å—Ç–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é : `echo $x` (—Ç–æ –µ—Å—Ç—å —Å–∞–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –≤—ã—Ä–∞–∂–∞–µ—Ç—Å—è —á–µ—Ä–µ–∑ –∑–Ω–∞–∫ –¥–æ–ª–ª–∞—Ä–∞). –ù–æ —Ç–∞–∫–∂–µ –º—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö —Å—Ä–∞–∑—É. –ù–∞–ø—Ä–∏–º–µ—Ä echo $x $name.
–ù–µ —Å—Ç–æ–∏—Ç –∑–∞–±—ã–≤–∞—Ç—å, —á—Ç–æ, –µ—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–µ–∫—Å—Ç –≤ –≤—ã–≤–æ–¥–µ, –Ω—É–∂–Ω–æ –ø–æ–º–µ—â–∞—Ç—å –≤—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –î–í–û–ô–ù–´–ï!!! –∫–∞–≤—ã—á–∫–∏ : `echo "Hi $name . I am $x y.o. And you?"`

–° –æ–¥–∏–Ω–∞—Ä–Ω—ã–º–∏ –∫–∞–≤—ã—á–∫–∞–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –Ω–µ –∞–∫—Ç–∏–≤–∏—Ä—É–µ—Ç—Å—è –∏ –º—ã –ø—Ä–æ—Å—Ç–æ –≤—ã–≤–µ–¥–µ–º —Ç–µ–∫—Å—Ç ( –Ω–∞–∑–≤–∞–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –≤–º–µ—Å—Ç–æ –µ—ë –∑–Ω–∞—á–µ–Ω–∏—è).

**–û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏**
–ú—ã –º–æ–∂–µ–º –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª–æ–º –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –∏ –≤ –¥—Ä—É–≥–∏—Ö —è–∑—ã–∫–∞—Ö –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, –ø—Ä–∞–≤–¥–∞ –µ—Å—Ç—å —Ñ–∏—à–∫–∏ –≤—Ä–æ–¥–µ —Ç–æ–≥–æ —á—Ç–æ –º—ã –Ω–µ –º–æ–∂–µ–º –ø—Ä–æ–ø–∏—Å–∞—Ç—å `new=$x+$y` –º—ã –ø–æ–ª—É—á–∏–º —Å—Ç—Ä–æ–∫–æ–≤—ã–π –æ—Ç–≤–µ—Ç `2+1`, —Ç–∞–∫ –∫–∞–∫ –∏–∑–Ω–∞—á–∞–ª—å–Ω–æ –≤—Å–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ —Å—Ç—Ä–æ–∫–æ–≤—ã–µ –ø–æ—ç—Ç–æ–º—É –º—ã –≤—ã—á–∏—Å–ª–∏—Ç–µ–ª—å–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –ø–æ–º–µ—â–∞–µ–º –≤ –¥–≤–µ —Å–∫–æ–±–∫–∏ `new=$(($x+$y))`.

## –ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏: –ø–æ–¥—Ä–æ–±–Ω–µ–µ
**–î–ª—è —Ä–∞–±–æ—Ç—ã —Å —á–∏—Å–ª–∞–º–∏ –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–ø–æ—Å–æ–±–æ–≤:**

```bash
x=5
y=3

# –°–ø–æ—Å–æ–± 1: $(( ... )) (–æ—Å–Ω–æ–≤–Ω–æ–π, —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è)
sum=$(($x + $y))       # 8
diff=$((x - y))        # 2. –û–±—Ä–∞—Ç–∏ –≤–Ω–∏–º–∞–Ω–∏–µ: –∑–Ω–∞–∫ $ –ø–µ—Ä–µ–¥ –∏–º–µ–Ω–∞–º–∏ –≤–Ω—É—Ç—Ä–∏ (()) –º–æ–∂–Ω–æ –æ–ø—É—Å—Ç–∏—Ç—å!
mult=$((x * y))        # 15
div=$((x / y))         # 1 (—Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –¥–µ–ª–µ–Ω–∏–µ!)

# –°–ø–æ—Å–æ–± 2: let
let "sum = x + y"      # 8
let "x++"              # –ò–Ω–∫—Ä–µ–º–µ–Ω—Ç. –¢–µ–ø–µ—Ä—å x=6

# –°–ø–æ—Å–æ–± 3: expr (—É—Å—Ç–∞—Ä–µ–≤—à–∏–π, –Ω–æ –≤—Å—Ç—Ä–µ—á–∞–µ—Ç—Å—è)
sum=`expr $x + $y`     # 8 (–∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –æ–±—Ä–∞—Ç–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏)
```

**–†–∞–±–æ—Ç–∞ —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏**

```bash

# –î–ª–∏–Ω–∞ —Å—Ç—Ä–æ–∫–∏

# –ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏—è (—Å–ª–æ–∂–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫)
```
